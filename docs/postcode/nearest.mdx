---
sidebar_label: "Nearest"
title: "Find Nearest Postcode"
---

Returns a list of postcodes nearest to the specified postcode, ordered by distance.

All standard postcode fields are returned for each result, plus a `distance` attribute (in metres from the specified postcode); if the input postcode is not found, the API returns a 404 response code.

For more details, see the [API documentation](/docs/api/nearest-postcode).

## Endpoint

```javascript
GET https://api.postcodes.io/postcodes/:postcode/nearest
```

## Example Request

```
https://api.postcodes.io/postcodes/NW19HZ/nearest
```

## Optional Query Parameters

| Parameter | Description                                                                   |
| --------- | ----------------------------------------------------------------------------- |
| `limit`   | Limits number of postcodes to return. Defaults to 10. Maximum allowed is 100. |
| `radius`  | Sets search radius in metres. Defaults to 100m. Maximum allowed is 2,000m.    |

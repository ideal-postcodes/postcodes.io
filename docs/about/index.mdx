Postcodes.io is an open sourced project maintained by Ideal Postcodes.
It is a free resource, allowing developers to search, reverse geocode and extract UK postcode and associated data.

## Data

| Dataset      | ONS Postcode Directory                                                                                                                                                                                                      | OS Open Names | Scottish Postcode Directory |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------- | --------------------------- |
| Release Date | February 2024                                                                                                                                                                                                               | January 2024  | January 2024                |
| Note         | Beyond the dates listed above, please be aware that the real-world dataset may change. Existing postcodes and places may be amended or discontinued. This information will not be captured until the next available update. |
